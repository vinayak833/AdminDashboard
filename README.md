# Admin Dashboard

A React-based Admin Dashboard designed to manage users, display statistics, and generate reports through a clean and responsive interface. The application demonstrates component-based architecture, state management, and dashboard-style UI development.

## Features

- Add and manage users
- Delete users
- Dashboard statistics cards
- Revenue tracking
- Reports section
- Responsive design
- Component-based structure

## Tech Stack

- React
- JavaScript (ES6)
- CSS
- Vite

## Installation

Clone the repository:

```bash
git clone https://github.com/vinayak833/AdminDashboard.git
```

Navigate to the project folder:

```bash
cd AdminDashboard
```

Install dependencies:

```bash
npm install
```

Run the application:

```bash
npm run dev
```

Open:

```text
http://localhost:5173
```

## Usage

1. Enter a user name.
2. Click "Add User".
3. View user statistics on the dashboard.
4. Monitor revenue and reports.
5. Delete users when needed.
